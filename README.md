# Portfolio

Highlights:

- Tech stack: MERN TailwindCSS
- you can send messages
- Global state management with Zustand

### Setup .env file

MONGO_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development

### Build the app
npm run build

### Start the app
npm run dev
